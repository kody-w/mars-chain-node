🔗 MARS CHAIN NODE

Decentralized verification node for the MARS virtual token economy.

API Endpoints

GET /api/state.json      — Chain state (head, circulating, verified count)
GET /api/verified.json   — All verified run attestations
GET /api/peers.json      — Known peer nodes
GET /api/consensus.json  — Latest cross-verification report

Submit a Cartridge

Export a .cartridge.json from the sim, then submit via PR to the submissions/ folder in this repo.
